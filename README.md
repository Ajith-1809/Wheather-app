A visually stunning, responsive weather dashboard featuring a **Glassmorphism** design system. This application delivers real-time weather updates, dynamic environmental backgrounds, and interactive data visualization.

## âœ¨ Features

*   **Dynamic Environments**: The interface adapts to current weather conditions (Sunny, Clear Night, Cloudy, Rain, Snow, Storm, Sunrise, Sunset).
*   **Glassmorphism UI**: Modern, translucent aesthetic with frosted glass effects.
*   **Real-time Data**: Current temperature, "feels like" temperature, and weather descriptions.
*   **Advanced Metrics**: Humidity, UV Index, Wind Speed & Direction.
*   **Celestial Info**: Sunrise, Sunset, and Moon Phase tracking.
*   **Interactive Charts**: Temperature trends visualized using **Chart.js**.
*   **Geolocation**: Automatic location detection for local weather.
*   **Unit Toggle**: Seamless switching between Celsius (Â°C) and Fahrenheit (Â°F).

### Installation

1.  **Clone the repository**

2.  **Navigate to the project directory**
